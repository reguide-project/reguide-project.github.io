# ReGuide

Source code for the [ReGuide project website](https://reguide-project.github.io).

**ReGuide: From Test-Time Guidance to Self-Improving Diffusion Policies**
Tzu-Hsiang Lin, Srinivas Shakkottai, Dileep Kalathil, P. R. Kumar — Texas A&M University

ReGuide is a self-improving diffusion-policy framework that treats guided rollouts as reusable
on-policy recovery data. Phase-Conditioned Guidance (PCG) generates corrective rollouts, which are
then absorbed back into the policy via fine-tuning (ReGuide-FT), from-scratch retraining
(ReGuide-FS), or their composition — yielding an iterative rollout–collect–train loop.
